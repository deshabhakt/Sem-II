      Steps:
      1. Load Image using cv2. imread()
      2. Create window using cv2. namedWindow()
      3. Resize the window using cv2. resizeWindow()
      4. Display Image using cv2. imshow()
      5. Wait for keyboard button press using cv2. waitKey()
      6. Exit window and destroy all windows using cv2. destroyAllWindows()
